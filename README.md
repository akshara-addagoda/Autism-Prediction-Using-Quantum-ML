# Autism Prediction Using Quantum Machine Learning

Developed by Akshara Addagoda

## 🧠 Overview
This project focuses on predicting Autism Spectrum Disorder (ASD) using a combination of classical machine learning and Quantum Machine Learning (QML) techniques. The system analyzes behavioral and medical input features to assist in early detection of autism.

## 🚀 Key Features
- Hybrid Quantum + Classical ML model
- User input-based prediction system
- Data preprocessing and feature handling
- Quantum circuit integration using Qiskit
- Real-time prediction output

## 🛠️ Technologies Used
- Python
- Qiskit (Quantum Computing)
- Scikit-learn
- NumPy & Pandas
- Matplotlib (for visualization)

## ⚙️ How It Works
1. User inputs relevant medical and behavioral data
2. Data is preprocessed and normalized
3. Features are encoded into a quantum circuit
4. Quantum ML model processes the data
5. Output predicts whether autism traits are present

## 📊 Use Case
This project can assist in early screening of autism, helping in quicker diagnosis and better intervention planning.
